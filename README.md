# EC React Practice

ECサイトの商品一覧を想定したフロントエンドアプリです。  
アパレルECの現場で想定されるUI構成を意識して制作しました。

## 制作目的

EC運用の経験を通じて、  
「課題を実装に落とし込む力」を強化するために制作しました。

実際の業務を想定し、
・商品一覧表示  
・カテゴリ切り替え  
・UIの視認性  
を意識しています。

## 使用技術

- React
- JavaScript
- CSS

## 機能

- 商品一覧表示
- カテゴリ切り替え
- モーダル表示

## 工夫した点

コンポーネント分割を意識し、  
再利用しやすい構造になるよう設計しました。
