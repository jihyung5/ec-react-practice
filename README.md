# 模擬ECサイト（React）

Reactを用いて作成した、アパレルECを想定した商品一覧画面のフロントエンド実装です。  
カテゴリ絞り込み・価格ソート・モーダル表示など、実務を想定したUI機能の実装を目的として制作しました。

## 使用技術

- React

---

## 主な機能

- 商品一覧表示
- カテゴリ絞り込み
- 価格ソート（昇順 / 降順）
- 商品詳細モーダル
- 表示件数の追加（もっと見るボタン）
- レスポンシブを意識したGridレイアウト

---

## 工夫した点

### UIロジックの分離
カテゴリ絞り込み → ソート → 表示件数制御の順にデータ加工を行い、  
状態管理をシンプルに保つ構成にしました。

### 再利用を意識したコンポーネント設計
商品カードをコンポーネント化し、UIの再利用性を意識しました。

### 実務を意識した機能実装
ECサイトでよく使われる以下の機能を実装しました。

- フィルタリング
- ソート
- モーダルUI
